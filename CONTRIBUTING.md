# Contributing to LumaTrace Public Portal

Thank you for your interest in improving the LumaTrace Public Developer Portal.

This repository is intentionally limited to API contracts, documentation, and integration examples. Proprietary core algorithms and backend infrastructure are not open for external contribution.

---

## Scope of Contributions

We welcome contributions related to:

- Documentation improvements or corrections
- Clarifications in API usage examples
- SDK example enhancements
- Suggestions for additional integration languages
- Improvements to OpenAPI specifications
- Enhancements to Postman collections

---

## How to Contribute

1. Open an Issue describing the proposed change.
2. Clearly explain the motivation and expected improvement.
3. Submit a Pull Request referencing the related Issue (if applicable).

All submissions are subject to internal review before acceptance.

---

## Important Notice

By submitting a contribution, you agree that it may be modified, merged, or rejected at the discretion of the maintainers.

Contributions must not include:
- Proprietary or confidential information
- Security-sensitive implementation details
- Any code unrelated to documentation or integration guidance
